

# Autonomous Vacuum Cleaning Robot (Vision-Based Dirt Detection)

## Overview
This project proposes an autonomous vacuum cleaning robot that uses computer vision and image processing techniques to detect dirt and perform intelligent cleaning in indoor environments.

## Key Features
- Vision-based dirt detection
- Image preprocessing (grayscale, blur, thresholding)
- Contour-based dirt identification
- Adaptive cleaning strategy
- Lighting condition testing

## Methodology
1. Image capture
2. Preprocessing using OpenCV
3. Thresholding and segmentation
4. Contour detection for dirt identification
5. Cleaning decision logic

## Technologies Used
- Python
- OpenCV
- NumPy
- Computer Vision techniques

## Research Contribution
This project explores integrating classical image processing with autonomous robotic cleaning systems for efficient dirt detection and targeted cleaning.

## Future Improvements
- Real-time implementation
- Integration with robotic hardware
- Deep learning-based detection
